⚠ ข้อมูลจะหายในเว็บเท่านั้น ไม่กระทบ Google Sheets

⚙ ตั้งค่าการเชื่อมต่อ

เชื่อมต่อกับ Google Apps Script ของคุณเพื่อโหลดข้อมูล

1เปิด Google Sheets → Extensions → Apps Script
2วางโค้ดจากไฟล์ Code.gs ทับทั้งหมด แล้วกด Save
3กด Deploy → New deployment → Type: Web app
4ตั้ง Execute as: Me / Who has access: Anyone
5Copy URL มาวางด้านล่าง
