# ASK External Data

ระบบจัดการข้อมูล External Drive สำหรับทีม Widewalker

🔗 **เข้าใช้งาน:** https://inv-ZEDER.github.io/WW-External-Data

---

## วิธีใช้งาน

เปิดลิงก์ด้านบนผ่านเบราว์เซอร์ได้เลย รองรับทั้งคอมและมือถือ

**ครั้งแรกที่เปิด** จะมี popup ขอ API URL — ให้ขอจากแอดมินครับ

---

## ฟีเจอร์

- 📁 ดูข้อมูล External แยกตามหมวด WW / CTV / BB / BN / TD
- 🔍 ค้นหาชื่อ External หรือ Project ได้ทันที
- ➕ เพิ่ม External และ Sub-project ใหม่ได้
- 🗂️ เพิ่มหมวดหมู่ใหม่ได้
- 🟢 แสดงสถานะ Storage Available / Unavailable

---

## ข้อมูลสำรอง

ข้อมูลทั้งหมดเก็บอยู่ใน Google Sheets หากเว็บใช้งานไม่ได้สามารถเปิด Sheets ได้โดยตรง
